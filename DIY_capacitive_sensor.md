# DIY Capacitive Sensor

Any conductive material could be used as a capacitive sensor, just by programming a micro-controller (such as Arduino UNO) and the corresponding libraries.

-----
### HARDWARE

- Arduino UNO

- Conductive material

- Jumper wires

-----
### LIBRARIES

- Open the Arduino IDE

- Open the Library Manager (book shelf icon at the left side of the Arduin IDE)

- Search for the following libraries at the search box (install both):

  - ADCTouch
  
  - CapacitiveSensor

- Optional: you can also download the libraries using this link. You will need to locate them in Arduino's main library folder.
